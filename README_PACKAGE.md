# reactstrap-sass

## Basic Usage

```js
import MyComponent from 'reactstrap-sass/cjs/ui/MyComponent' // commonjs

import MyComponent from 'reactstrap-sass/es/ui/MyComponent' // ES modules
```

```html
<MyComponent onClick={() => {}}>Test</MyComponent>
```
